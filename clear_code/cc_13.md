# 1.
```
class DynArray:

    def __init__(self):
        self.count = 0
```
Пример максимально раннего связывания. Здесь я задаю изначальное значение параметра класса, содержащего размер динамического массива. На мой взгляд, это наглядно и легко для понимания, если этот код будет читаться и использоваться кем-то другим. Константа здесь не подходит, так как значение будет меняться в ходе заполнения массива, а переменная снизила бы читаемость кода. Насколько понимаю, здесь 0 не является магическим числом.

# 2.
```
class SocialLink: # Сюжетное ответвление для установления отношений с персонажем
    
    def __init__(self, sl_name, sl_arcana, sl_current_level, sl_current_date, sl_deadline):
        self.name = sl_name  # Имя персонажа
```
Пример связывания во время компиляции кода. Параметр класса, значение которого задается переменной, поскольку оно может прнимать различные значения. Так как значение имени персонажа не известно заранее и будет инициировано в процессе игры, то это время связывания показалось мне уместным.

# 3.
```
def random_animal(ams: list[Animal]):
    ams = []
    for i in range(500):
        coin_toss = random.randint(1,2)
        if coin_toss == 1:
            animal = Cat()
            ams.append(animal)
        else:
            animal = Bird()
            ams.append(animal)
    return ams
```
Пример связывание во время выполнения программы. Результат содержимого переменной с животным будет зависеть от результата работы программы, поэтому запись объекта класса внутри нее имеет смысл делать именно на этом уровне.
