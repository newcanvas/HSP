// В целом, даю индексам счетчиков нормальные названия, иногда использую более выразительные, чем однобуквенные, например, lvl. Слишном длинными их не делаю. Дальше пример того, как в цикле из задачи про Мастера Ключей можно заменить однобуквенные названия на более длинные, зато отражающие логику происходящего:

for every_door in range(k):
    for every_second_door in range(k):
        if (every_second_door+1) % (every_door+1) == 0:
            doors[every_second_door] = not doors[every_second_door]
